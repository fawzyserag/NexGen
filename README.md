# NexGen Solutions — Landing Page

## Tech Stack
- Frontend: Angular 21
- Backend: .NET 9 Web API

## How to Run

### Backend
cd LandingPageAPI
dotnet run --urls "http://localhost:5000"

### Frontend
cd landing-page-frontend
npm install
ng serve

## API Endpoints
POST /api/contact  — Submit contact form
GET  /api/contact/health — Health check
